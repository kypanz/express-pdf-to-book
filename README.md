# express-pdf-to-book

Simple backend to generate audios based on pdf books

## Configure

You gonna need to create a `env` file and add the next things

```
MODE_TEST='false'
HTTPS='false'
EMAIL_ENABLED='false'
EMAIL_HOST='example.smtp'
EMAIL_USER='example@gmail.com'
EMAIL_PASSWORD=''
URL_FRONTEND='https://example.com'
```
